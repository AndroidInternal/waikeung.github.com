1. 文件目录操作

1.1. ls 命令

ls命令是linux上最常用的命令。估计也是到目前为止我用的次数最多的命令。ls命令做一件很简单的事情，就是列出目录中文件和文件夹的清单。用ls不仅可以查看linux下的文件，而且可以查看文件的权限，文件一些信息等等。

  1. 命令的功能
    列出查看目录下的文件和子目录。
  2. 命令格式
    ls [选项] [目录名]
  3. 常用参数 
        -a, --all
            列出目录下的所有文件，包括隐藏的文件(以.开头的文件为隐藏文件)
    -A, --almost-all
            同-a，但不列出"."和".."
    -d, --directory
            将目录像文件一样显示，而不是显示其下的文件。
    -F
            列出的文件名称后面加上一符号，如可执行文件加'*',目录加'/'
    -i, --inode
            列出每个文件的inode号
    -r, --reverse
            依相反次序排序
    -R, --recursive
            同时列出子目录
    -S
            根据文件大小排序
    -t
            以文件修改时间排序
    -l
            除了列出文件名之外，还将文件的权限，所有者，文件大小等信息详细列出来。
  4. 常用范例：
    1. 列出目录下的所有文件和子目录的详细资料 ls -lR或ls -l -R
    2. 列出当前目录下所有以't'开头的文件及子目录详情ls -l t*
    3. 只列出当前目录下的子目录 ls -F | grep /$
    4. 计算当前目录下的文件数和目录数 ls -l * | grep "^-" |wc -l ls -l * | grep "^d" | wc -l
    5. 指定文件时间输出格式
      ls -tl --time-style=ful-iso 
      waikeung@waikeung-ThinkPad:~$ ls -tl --time-style=full-iso
total 104
drwxr-xr-x  3 waikeung waikeung  4096 2014-02-05 15:48:58.907178957 +0800 Downloads

      ls -tl --time-style=long-iso 
      waikeung@waikeung-ThinkPad:~$ ls -tl --time-style=long-iso
total 104
drwxr-xr-x  3 waikeung waikeung  4096 2014-02-05 15:48 Downloads

1.2. cd 命令

cd命令也是linux上最常用的命令之一。cd的命令很简单。

  1. 命令功能
    切换当前目录到其它目录
  2. 命令格式
    cd [目录名]
  3. 常用范例
    1. 进入系统根目录/ cd /
    2. 进入上一层目录 cd ../
    3. 进入当前用户根目录 cd或cd ~
    4. 返回进入此目录之前所在的目录 
      waikeung@waikeung-ThinkPad:~$ cd /
waikeung@waikeung-ThinkPad:/$ pwd
/
waikeung@waikeung-ThinkPad:/$ cd 
waikeung@waikeung-ThinkPad:~$ pwd
/home/waikeung
waikeung@waikeung-ThinkPad:~$ cd -
/
waikeung@waikeung-ThinkPad:/$ cd -
/home/waikeung
    5. 把上一个命令的参数作为cd参数使用 
      waikeung@waikeung-ThinkPad:/tmp$ mkdir temp
waikeung@waikeung-ThinkPad:/tmp$ cd !$
cd temp
waikeung@waikeung-ThinkPad:/tmp/temp$ pwd
/tmp/temp
waikeung@waikeung-ThinkPad:/tmp/temp$

1.3. pwd 命令

pwd命令是用来查看你当前的工作目录的完整路径的。当你不知道你在哪时可以用这个命令来查看。到目前我不怎么用到，因为我基本上都很清楚我自己在哪。

  1. 命令功能
    查看当前工作目录的完整路径
  2. 命令格式
    pwd [参数]
  3. 常用参数
    一般的情况下，该命令都不需要用到参数。如果目录是链接时，pwd -P显示出实际的路径，而非链接的路径。
  4. 常用示例
    1. 查看当前的工作目录 pwd
    2. 目录连接链接时，pwd -P显示实际的路径。 
      waikeung@waikeung-ThinkPad:~$ cd /opt/eclipse/jre/
waikeung@waikeung-ThinkPad:/opt/eclipse/jre$ pwd
/opt/eclipse/jre
waikeung@waikeung-ThinkPad:/opt/eclipse/jre$ pwd -P
/opt/jdk1.7.0_45/jre
waikeung@waikeung-ThinkPad:/opt/eclipse/jre$

1.4. mkdir 命令

linux中创建指定名称的目录。要求创建目录的用户在当前目录中具有写权限，同时指定的目录名不能是当前目录中已有的目录。

  1. 命令功能
    创建指定的目录名的目录。
  2. 命令格式
    mkdir [参数] 目录
  3. 常用参数
    -m, --mode=MODE
    set file mode (as in chmod), not a=rwx - umask
    -p, --parents
    可以是一个路径名称，若此时路径中有某些目录不存在，系统会自动建立那些还不存在的目录。
    -v, --verbose
    创建新目录时显示信息
  4. 常用示例
    1. 创建一个空目录 mkdir test
    2. 递归创建多个目录 
      waikeung@waikeung-ThinkPad:/tmp$ mkdir -p test/test1
waikeung@waikeung-ThinkPad:/tmp$ cd test/test1/
waikeung@waikeung-ThinkPad:/tmp/test/test1$
    3. 创建权限为777的目录 
      waikeung@waikeung-ThinkPad:/tmp/test$ mkdir -m 777 a
waikeung@waikeung-ThinkPad:/tmp/test$ ls -l
total 8
drwxrwxrwx 2 waikeung waikeung 4096 Feb  5 20:10 a

1.5. rmdir 命令

rmdir命令是删除一个空目录。一个目录被删除之前必须是空的，删除目录必须对其父目录具有写的权限。

  1. 命令功能
    删除空目录
  2. 命令格式
    rmdir [参数] 目录
  3. 常用参数
    -p
    递归的删除目录，当子目录删除后父目录为空时，也删除父目录。
    -v, --verbose
    显示指令执行过程
  4. 常用示例
    1. rmdir不能删除非空目录 
      waikeung@waikeung-ThinkPad:/tmp$ rmdir test
rmdir: failed to remove `test': Directory not empty

1.6. rm 命令

linux中删除文件的命令为rm。rm是最常用的命令之一，但也是危险的命令。它可以将某个目录下的所有文件及子目录全部删除。对于链接文件，只是删除了链接，原有文件保持不变。

  1. 命令功能
    删除一个目录下的文件或多个文件或多个目录。
  2. 命令格式
    rm [参数] 文件
  3. 常用参数
    -f, --force
    忽略不存在的文件,不给出提示
    -i，--interactive
    进行交互式删除
    -r, -R, --recursive
    将列出的全部目录和子目录均递归删除
    -v, --verbose
    显示指令执行过程
  4. 命令示例
    1. 删除文件file，系统会选询问 
      waikeung@waikeung-ThinkPad:/tmp/temp$ ls
log
waikeung@waikeung-ThinkPad:/tmp/temp$ rm -i log 
rm: remove regular empty file `log'? y
waikeung@waikeung-ThinkPad:/tmp/temp$ ls
waikeung@waikeung-ThinkPad:/tmp/temp$
    2. 强行删除文件或目录 
      waikeung@waikeung-ThinkPad:/tmp/temp$ ls -l
total 12
drwxrwxr-x 2 waikeung waikeung 4096 Feb  5 20:37 1
drwxrwxr-x 2 waikeung waikeung 4096 Feb  5 20:37 2
drwxrwxr-x 2 waikeung waikeung 4096 Feb  5 20:37 3
waikeung@waikeung-ThinkPad:/tmp/temp$ cd ../
waikeung@waikeung-ThinkPad:/tmp$ rm -r temp
rm: descend into directory `temp'? y
rm: remove directory `temp/1'? y
rm: remove directory `temp/2'? y
rm: remove directory `temp/3'? y
rm: remove directory `temp'? y
waikeung@waikeung-ThinkPad:/tmp$
    3. rm -rf会删除目录中所有的文件和子目录，且不会询问。 ！！！危险的命令

1.7. cp 命令

cp命令是用来复制文件或者目录的。

  1. 命令功能
    将源文件复制到目标文件，或将多个源文件复制到目录。
  2. 命令格式
    cp [参数] [-T] 源 目标
    cp [参数] 源 目录
    cp [参数] -t 目录 源
  3. 常用参数
    -a, --archive
    等于 -dR --preserve=all
    -d
    等于 --no-dereference --preserve=links,拷贝时保留链接
    -f, --force
    如果目标文件无法打开则将其删除并重试(当 -n 选项存在时则不需要再选此项)
    -i, --interactive
    交互式复制(使用-n选项失效)
    -l, --link
    建立硬链接而不是复制
    -n, --no-clobber
    不要覆盖已经存在的文件
    -p
    复制源文件内容后，还将其修改时间和访问权限也复制到新文件
    -R, -r, --recursive
    复制目录及目录内的所有项目
    -s, --symbolic-link
    复制成符号链接文件
    -T, --no-target-directory
    把目标当成一个普通文件
    -u, --update
    只会在源文件的修改时间较目标文件新时，才会复制
  4. 常用示例
    1. 将文件file1复制成file2，复制后名称被改为file2 cp -i file1 file2 或 cp file1 file2
    2. 将文件file1复制到dir1目录下，复制后的名称仍为file1 cp -i file1 dir1 或 cp file1 dir1
    3. 将目录dir1复制到dir2目录下 cp -r dir1 dir2
    4. 将目录dir1下所有文件包括子目录，都复制到dir2目录下 cp -r dir1/*.* dir2

1.8. mv 命令

linux下用来移动文件或修改文件名的命令

  1. 命令功能
    视mv命令中第二个参数类型的不同（是目标文件还是目标目录），mv命令将文件重命名或将其移至一个新的目录中。当第二个参数类型是文件时，mv命令完成文件重命名，此时，源文件只能有一个（也可以是源目录名），它将所给的源文件或目录重命名为给定的目标文件名。当第二个参数是已存在的目录名称时，源文件或目录参数可以有多个，mv命令将各参数指定的源文件均移至目标目录中。在跨文件系统移动文件时，mv先拷贝，再将原有文件删除，而链至该文件的链接也将丢失。
  2. 命令格式
    mv [参数] 源文件或目录 目标文件或目录
