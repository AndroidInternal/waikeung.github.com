文件目录操作

ls 命令

ls命令是linux上最常用的命令。估计也是到目前为止我用的次数最多的命令。ls命令做一件很简单的事情，就是列出目录中文件和文件夹的清单。用ls不仅可以查看linux下的文件，而且可以查看文件的权限，文件一些信息等等。

  1. 命令的功能
    列出查看目录下的文件和子目录。
  2. 命令格式
    ls [选项] [目录名]
  3. 常用参数 
        -a, --all
            列出目录下的所有文件，包括隐藏的文件(以.开头的文件为隐藏文件)
    -A, --almost-all
            同-a，但不列出"."和".."
    -d, --directory
            将目录像文件一样显示，而不是显示其下的文件。
    -F
            列出的文件名称后面加上一符号，如可执行文件加'*',目录加'/'
    -i, --inode
            列出每个文件的inode号
    -r, --reverse
            依相反次序排序
    -R, --recursive
            同时列出子目录
    -S
            根据文件大小排序
    -t
            以文件修改时间排序
    -l
            除了列出文件名之外，还将文件的权限，所有者，文件大小等信息详细列出来。
  4. 常用范例：
    1. 列出目录下的所有文件和子目录的详细资料 ls -lR或ls -l -R
    2. 列出当前目录下所有以't'开头的文件及子目录详情ls -l t*
    3. 只列出当前目录下的子目录 ls -F | grep /$
    4. 计算当前目录下的文件数和目录数 ls -l * | grep "^-" |wc -l ls -l * | grep "^d" | wc -l
    5. 指定文件时间输出格式
      ls -tl --time-style=ful-iso 
      waikeung@waikeung-ThinkPad:~$ ls -tl --time-style=full-iso
total 104
drwxr-xr-x  3 waikeung waikeung  4096 2014-02-05 15:48:58.907178957 +0800 Downloads

      ls -tl --time-style=long-iso 
      waikeung@waikeung-ThinkPad:~$ ls -tl --time-style=long-iso
total 104
drwxr-xr-x  3 waikeung waikeung  4096 2014-02-05 15:48 Downloads

cd 命令

cd命令也是linux上最常用的命令之一。cd的命令很简单。

  1. 命令功能
    切换当前目录到其它目录
  2. 命令格式
    cd [目录名]
  3. 常用范例
    1. 进入系统根目录/ cd /
    2. 进入上一层目录 cd ../
    3. 进入当前用户根目录 cd或cd ~
    4. 返回进入此目录之前所在的目录 
      waikeung@waikeung-ThinkPad:~$ cd /
waikeung@waikeung-ThinkPad:/$ pwd
/
waikeung@waikeung-ThinkPad:/$ cd 
waikeung@waikeung-ThinkPad:~$ pwd
/home/waikeung
waikeung@waikeung-ThinkPad:~$ cd -
/
waikeung@waikeung-ThinkPad:/$ cd -
/home/waikeung
    5. 把上一个命令的参数作为cd参数使用 
      waikeung@waikeung-ThinkPad:/tmp$ mkdir temp
waikeung@waikeung-ThinkPad:/tmp$ cd !$
cd temp
waikeung@waikeung-ThinkPad:/tmp/temp$ pwd
/tmp/temp
waikeung@waikeung-ThinkPad:/tmp/temp$

pwd 命令

pwd命令是用来查看你当前的工作目录的完整路径的。当你不知道你在哪时可以用这个命令来查看。到目前我不怎么用到，因为我基本上都很清楚我自己在哪。

  1. 命令功能
    查看当前工作目录的完整路径
  2. 命令格式
    pwd [参数]
  3. 常用参数
    一般的情况下，该命令都不需要用到参数。如果目录是链接时，pwd -P显示出实际的路径，而非链接的路径。
  4. 常用示例
    1. 查看当前的工作目录 pwd
    2. 目录连接链接时，pwd -P显示实际的路径。 
      waikeung@waikeung-ThinkPad:~$ cd /opt/eclipse/jre/
waikeung@waikeung-ThinkPad:/opt/eclipse/jre$ pwd
/opt/eclipse/jre
waikeung@waikeung-ThinkPad:/opt/eclipse/jre$ pwd -P
/opt/jdk1.7.0_45/jre
waikeung@waikeung-ThinkPad:/opt/eclipse/jre$

mkdir 命令

linux中创建指定名称的目录。要求创建目录的用户在当前目录中具有写权限，同时指定的目录名不能是当前目录中已有的目录。

  1. 命令功能
    创建指定的目录名的目录。
  2. 命令格式
    mkdir [参数] 目录
  3. 常用参数 
      -m, --mode=MODE
        set file mode (as in chmod), not a=rwx - umask
  -p, --parents
        可以是一个路径名称，若此时路径中有某些目录不存在，系统会自动建立那些还不存在的目录。
  -v, --verbose
        创建新目录时显示信息
  4. 常用示例
    1. 创建一个空目录 mkdir test
    2. 递归创建多个目录 
      waikeung@waikeung-ThinkPad:/tmp$ mkdir -p test/test1
waikeung@waikeung-ThinkPad:/tmp$ cd test/test1/
waikeung@waikeung-ThinkPad:/tmp/test/test1$
    3. 创建权限为777的目录 
      waikeung@waikeung-ThinkPad:/tmp/test$ mkdir -m 777 a
waikeung@waikeung-ThinkPad:/tmp/test$ ls -l
total 8
drwxrwxrwx 2 waikeung waikeung 4096 Feb  5 20:10 a

rmdir 命令

rmdir命令是删除一个空目录。一个目录被删除之前必须是空的，删除目录必须对其父目录具有写的权限。

  1. 命令功能
    删除空目录
  2. 命令格式
    rmdir [参数] 目录
  3. 常用参数 
      -p
        递归的删除目录，当子目录删除后父目录为空时，也删除父目录。
  -v,  --verbose
        显示指令执行过程
  4. 常用示例
    1. rmdir不能删除非空目录 
      waikeung@waikeung-ThinkPad:/tmp$ rmdir test
rmdir: failed to remove `test': Directory not empty

rm 命令

linux中删除文件的命令为rm。rm是最常用的命令之一，但也是危险的命令。它可以将某个目录下的所有文件及子目录全部删除。对于链接文件，只是删除了链接，原有文件保持不变。

  1. 命令功能
    删除一个目录下的文件或多个文件或多个目录。
  2. 命令格式
    rm [参数] 文件
  3. 常用参数 
      -f, --force
        忽略不存在的文件,不给出提示
  -i，--interactive
        进行交互式删除
  -r, -R, --recursive
        将列出的全部目录和子目录均递归删除
  -v, --verbose
        显示指令执行过程
  4. 命令示例
    1. 删除文件file，系统会选询问 
      waikeung@waikeung-ThinkPad:/tmp/temp$ ls
log
waikeung@waikeung-ThinkPad:/tmp/temp$ rm -i log 
rm: remove regular empty file `log'? y
waikeung@waikeung-ThinkPad:/tmp/temp$ ls
waikeung@waikeung-ThinkPad:/tmp/temp$
    2. 强行删除文件或目录 
      waikeung@waikeung-ThinkPad:/tmp/temp$ ls -l
total 12
drwxrwxr-x 2 waikeung waikeung 4096 Feb  5 20:37 1
drwxrwxr-x 2 waikeung waikeung 4096 Feb  5 20:37 2
drwxrwxr-x 2 waikeung waikeung 4096 Feb  5 20:37 3
waikeung@waikeung-ThinkPad:/tmp/temp$ cd ../
waikeung@waikeung-ThinkPad:/tmp$ rm -r temp
rm: descend into directory `temp'? y
rm: remove directory `temp/1'? y
rm: remove directory `temp/2'? y
rm: remove directory `temp/3'? y
rm: remove directory `temp'? y
waikeung@waikeung-ThinkPad:/tmp$
    3. rm -rf会删除目录中所有的文件和子目录，且不会询问。 ！！！危险的命令

cp 命令

cp命令是用来复制文件或者目录的。

  1. 命令功能
    将源文件复制到目标文件，或将多个源文件复制到目录。
  2. 命令格式
    cp [参数] [-T] 源 目标
    cp [参数] 源 目录
    cp [参数] -t 目录 源
  3. 常用参数 
      -a, --archive
        等于 -dR --preserve=all
  -d
        等于 --no-dereference --preserve=links,拷贝时保留链接
  -f, --force
        如果目标文件无法打开则将其删除并重试(当 -n 选项存在时则不需要再选此项)
  -i, --interactive
        交互式复制(使用-n选项失效)
  -l, --link
        建立硬链接而不是复制
  -n, --no-clobber
        不要覆盖已经存在的文件
  -p
        复制源文件内容后，还将其修改时间和访问权限也复制到新文件
  -R, -r, --recursive
        复制目录及目录内的所有项目
  -s, --symbolic-link
        复制成符号链接文件
  -T, --no-target-directory
        把目标当成一个普通文件
  -u, --update
        只会在源文件的修改时间较目标文件新时，才会复制
  4. 常用示例
    1. 将文件file1复制成file2，复制后名称被改为file2 cp -i file1 file2 或 cp file1 file2
    2. 将文件file1复制到dir1目录下，复制后的名称仍为file1 cp -i file1 dir1 或 cp file1 dir1
    3. 将目录dir1复制到dir2目录下 cp -r dir1 dir2
    4. 将目录dir1下所有文件包括子目录，都复制到dir2目录下 cp -r dir1/*.* dir2

mv 命令

linux下用来移动文件或修改文件名的命令

  1. 命令功能
    视mv命令中第二个参数类型的不同（是目标文件还是目标目录），mv命令将文件重命名或将其移至一个新的目录中。当第二个参数类型是文件时，mv命令完成文件重命名，此时，源文件只能有一个（也可以是源目录名），它将所给的源文件或目录重命名为给定的目标文件名。当第二个参数是已存在的目录名称时，源文件或目录参数可以有多个，mv命令将各参数指定的源文件均移至目标目录中。在跨文件系统移动文件时，mv先拷贝，再将原有文件删除，而链至该文件的链接也将丢失。
  2. 命令格式
    mv [参数] 源文件或目录 目标文件或目录
  3. 常用参数 
      -f, --force
        如果目标文件爱你存在，直接覆盖。
  -i, --interactive
        交互式移动或重命名。
  -t, --target-directory=DIRECTORY
        把所有文件移动到目标目录，适用于移动多文件。
  4. 常用示例
    1. 文件改名 mv file1 file2 将file1重命名为file2。
    2. 移动文件 mv file1 dir1 将file1移动到dir1目录中。
    3. 将file1，file2，file3移动到dir1目录中。 mv -t dir1 file1 file2 file3
    4. 目录的移动 mv dir1 dir2 若dir2不存在，则dir1重命名为dir2，若dir2存在，则dir1移动到dir2中。

cat 命令

cat命令可以连接文件和将文件打印到标准输出中。

  1. 命令功能
    cat命令有三大功能
    1. 一次显示整个文件：cat filename
    2. 从键盘创建一个新文件：cat > filename 只能创建新文件，不能编译以后文件。
    3. 将一个文件合并成一个文件：cat file1 file2 file3 > file4
  2. 命令格式
    cat [参数] 文件
  3. 常用参数 
      -A, --show-all
        等价与 -vET
  -b, --number-nonblank
        对非空的行输出行编号。
  -E, --show-ends
        在每行结束处显示$
  -n, --number
        对所有输出行显示行编号
  -s, --squeeze-blank
        多行空白行替换为一行空白行
  -T, --show-tabs
        将TAB键显示为^I
  -v, --show-nonprinting
        用^和M-引用，除了LFD和TAB
  4. 常用示例
    1. 将一个文件打印到标准输出 cat filename
    2. 从键盘创建一个新文件 
      waikeung@waikeung-ThinkPad:/tmp$ cat > hi <<EOF
> Hello
> I'm here!
> PWD=$(pwd)
> EOF
waikeung@waikeung-ThinkPad:/tmp$ cat hi
Hello
I'm here!
PWD=/tmp
waikeung@waikeung-ThinkPad:/tmp$
    3. 将file1 file2 合并成 file 
      waikeung@waikeung-ThinkPad:/tmp$ cat file1
one
two
three
waikeung@waikeung-ThinkPad:/tmp$ cat file2
1
2
3
waikeung@waikeung-ThinkPad:/tmp$ cat file1 file2 > file
waikeung@waikeung-ThinkPad:/tmp$ cat file
one
two
three
1
2
3
waikeung@waikeung-ThinkPad:/tmp$

more 命令

more命令有点像cat命令，但有不同，cat是直接全部显示在屏幕上，而more是一页一页显示的，通过空格键(space)就往下一页。b键前往上一页，只对文件有效，对管道无效。

  1. 命令功能
    查看文件的内容，可以按页来查看文件，还支持直接搜索字符串的功能。
  2. 命令格式
    more [参数] [文件]
  3. 常用参数 
      +n
        从第n行开始显示
  -n
        定义屏幕大小为n行
  -c
        清屏，然后开始显示
  -d
        显示提示"Press space to continue，’q’ to quit"，同时禁用响铃
  -p
        通过清除窗口而不是滚屏来对文件进行换页
  -s
        把连续的多个空行显示为一行
  4. 常用操作命令
    /string 查找string
    Enter 向下滚动n行，需要定义，默认为1行
    Space 向下滚动一屏
    = 输出当前行号
    b 返回上一屏
    :f 输出当前文件名和当前行号
    V 调用vi编辑器
    ！cmd 调用shell，并执行cmd
    q 退出more
  5. 常用示例
    1. 显示文件中第5行起的内容 more +5 file
    2. 配合管道适用 
      waikeung@waikeung-ThinkPad:/$ ls -l | more -5
total 87
drwxr-xr-x   2 root root  4096 Jan 17 10:08 bin
drwxr-xr-x   4 root root  3072 Jan 17 10:12 boot
drwxr-xr-x   2 root root  4096 Dec 10 18:00 cdrom
drwxr-xr-x  18 root root  4460 Feb  9 07:32 dev

less 命令

less也是查看文件，对文件分页显示的工具。与more相比，less的功能更加强大。

  1. 命令功能
    与more类似。但less能随意浏览文件，还能向前向后搜索。
  2. 命令格式
    less [参数] 文件
  3. 常用参数 
      -b
        <缓冲区大小> 设置缓冲区的大小
  -e
        当文件显示结束后，自动离开
  -f
        强迫打开特殊文件，例如外围设备代号、目录和二进制文件
  -g
        只标志最后搜索的关键词
  -i
        忽略搜索时的大小写
  -m
        显示类似more命令的百分比
  -N
        显示每行的行号
  -o
        <文件名> 将less 输出的内容在指定文件中保存起来
  -Q
        不使用警告音
  -s
        显示连续空行为一行
  -S
        行过长时间将超出部分舍弃
  -x
        <数字> 将“tab”键显示为规定的数字空格
  4. 常用操作命令
    /string： 向下搜索“string”的功能
    ?string： 向上搜索“string”的功能
    n： 重复前一个搜索（与 / 或 ? 有关）
    N： 反向重复前一个搜索（与 / 或 ? 有关）
    b 向后翻一页
    d 向后翻半页
    h 显示帮助界面
    Q 退出less 命令
    u 向前滚动半页
    y 向前滚动一行
    空格键 滚动一行
    回车键 滚动一页
    [pagedown]: 向下翻动一页
    [pageup]： 向上翻动一页
  5. 常用示例
    1. 查看文件 less file
    2. ps查看进程用less分页显示 ps aux | less

head 命令

head用来显示文件的开头。

  1. 命令功能
    显示文件的开头到标准输出。默认显示10行。
  2. 命令格式
    head [参数] [文件]
  3. 常用参数 
      -c, --bytes=[-]k
        显示字节数
  -n, --lines=[-]k
        显示行数
  -q, --quiet, --silent
        不显示文件名
  -v, --verbose
        显示文件名
  4. 常用示例
    1. 显示文件前5行 head -n 5 file
    2. 显示文件前32个字节 head -c 32 file
    3. 显示文件除了最后5行 head -n -5 file
    4. 显示文件除了最后32个字节 head -c -32 file

  4. 常用示例

    2. 从第5行开始显示文件 tail -n -5 file

  1. 命令功能

  2. 命令格式
    which 可执行文件名称
  3. 常用参数
    一般我都不怎么用到参数
  4. 常用示例
    1. 查找命令，显示命令路径 which command
    2. which是找不到bash的内建命令的,比如cd。因为which默认是找PATH内的命令。
