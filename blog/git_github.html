Git/GitHub使用

Git的基本配置

git config --global user.name   # Your Name Here
git config --global user.email  # Your_email@example.com

查看已有的配置 git config --list

Git的基本使用

git remote add origin git_address

git push -u orgin master

git clone git_address

分支(branch)

创建分支

  • 创建分支并同时切换到新建分支上 git checkout -b branch_name
  • 先创建一个分支再切换到新分支上 git branch branch_name git checkout branch_name

查看分支

  • 查看该项目的所有分支 git branch
  • 通过git checkout master切换到主分支上

合并分支

git add .git # 添加新增的文件
git commit -m "add new feature"
git checkout master # 切换到主分支
git merge branch_name # 将新分支的内容合并到主分支上，主分支和新分支就一样了

删除分支：在主分支上

  • 删除已经合并到主分支的branch git branch -d branch_name
  • 删除branch分支，不管有没有合并 git branch -D branch_name

实用：回到项目之前的提交版本

  • 查看提交记录 git log
  • 想回到之前的版本，简单的用提交的ID就可以了，前几位ID（可以区分其他ID号，即其他版本就可以了）git checkout 085bb3bcb
  • 将之前这个版本变成一个新的独立分支 git checkout -b branch_name 085bb3bcb
